# 诡辩术

**修辞的操纵, 逻辑的谬误**

## "晓之以理"

<p><img align="right" alt="1-诉诸匿名权威.png" src="https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/1-诉诸匿名权威.png" width="20%"></p>

### 诉诸匿名权威  
**引用来源不详的"砖家"、"研究"或某一群体(比如"科学家")以声称某观点是正确的**  
"他们说要花7年才能消化一片口香糖"

****

<p><img align="right" alt="2-诉诸（可疑）权威.png" src="https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/2-诉诸（可疑）权威.png" width="20%"></p>

### 诉诸(可疑)权威  
**因为某个无实无信的"专家"说某件事是真的, 因而断言确实如此**  
"超过400位杰出的科学家与工程师对全球变暖持争议态度。"

****

![3-诉诸常规.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/3-诉诸常规.png){ align=right width="20%" }
### 诉诸常规  
**因为常见, 所以正确**  
"这家银行有些贪污腐败方面的问题。但在这里发生的事不是在哪家银行都在发生的嘛。"

****

![4-诉诸无知.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/4-诉诸无知.png){ align=right width="20%" }
### 诉诸无知  
**某一观点是正确的, 仅仅因为它没有被证伪(或某观点是错误的, 仅仅因为它尚未被证实)。**  
"没人能证明有上帝。所以没有上帝。"

****

![5-诉诸怀疑.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/5-诉诸怀疑.png){ align=right width="20%" }
### 诉诸怀疑  
**因为某件事听起来不可信, 所以一定不是真的。**  
"眼睛是超级复杂的生物机械之作, 有千万个精密联系的不减。如果没有一位睿智的设计师, 这怎么可能存在?"

****

![6-身价逻辑.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/6-身价逻辑.png){ align=right width="20%" }
### 身价逻辑  
**如果某人很有钱, 或者某样东西很贵, 那么这就对某一论断的真实性造成了影响。**  
"如果这玩意更贵的话, 那它一定更好。"

****

![7-求新逻辑.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/7-求新逻辑.png){ align=right width="20%" }
### 求新逻辑  
**因为是最新的, 所以更好**  
"太棒了! 最新的操作系统会让我的电脑更快的..."

****

![8-诉诸主流.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/8-诉诸主流.png){ align=right width="20%" }
### 诉诸主流  
**认定某件事是真的, 因为大多数人都这么相信。**  
"喝牛奶能使你骨骼强健"

****

![9-诉诸传统.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/9-诉诸传统.png){ align=right width="20%" }
### 诉诸传统  
**声称某件事是正确的, 因为(很显然)一直以来都是这样。**  
"婚姻是男与女的结合。因此同性婚姻毫无依据。"

****

![10-诉诸概率.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/晓之以理/10-诉诸概率.png){ align=right width="20%" }
### 诉诸概率  
**相信因为某件事情可能发生, 所以必然会发生。**  
"宇宙里有数不清的星系, 无数多的星星, 一定有另一颗行星孕育了智慧生命。"

****

## "动之以情"

![1-掩耳盗铃.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/1-掩耳盗铃.png){ align=right width="20%" }
### 掩耳盗铃  
**声称某个观点是错误的, 因为你不愿意相信那所意味的事实。**  
"他不可能就是为了骗我的钱。他说过他爱我的, 他一定是遭到了什么变故。"

****

![2-诉诸恐惧.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/2-诉诸恐惧.png){ align=right width="20%" }
### 诉诸恐惧  
**煽动对一方的恐惧与偏见, 从而进行论证**  
"在你意识到之前, 清真寺的数目就会超过教堂了。"

****

![3-诉诸陷媚.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/3-诉诸陷媚.png){ align=right width="20%" }
### 诉诸陷媚  
**给毫无根据的论点裹上糖衣炮弹, 让人不自觉地全盘接受**  
"聪明又有洞察力的读者当然在读到这样的谬误时就能马上发觉"

****

![4-诉诸自然.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/4-诉诸自然.png){ align=right width="20%" }
### 诉诸自然  
**通过与"至善"的自然界的对比, 来使你的观点看起来更站得住脚。**  
"同性恋当然违背天性。你看不到同性动物交配吧。"

****

![5-诉诸同情.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/5-诉诸同情.png){ align=right width="20%" }
### 诉诸同情  
**唤起人们的怜悯之心, 以摇动对手。**  
"前独裁者已垂垂老矣, 濒临末年。让他为这些指控接受审判实在不应该。"

****

![6-诉诸荒谬.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/6-诉诸荒谬.png){ align=right width="20%" }
### 诉诸荒谬  
**将对手的观点以谬误的形式表示出来以进行打击**  
"对上帝的忠诚就如同相信有圣诞老人和牙仙一样。"

****

![7-诉诸仇恨.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/7-诉诸仇恨.png){ align=right width="20%" }
### 诉诸仇恨  
**因为个人偏见而对某一看法不屑一顾**  
"富二代搞慈善? 算了吧, 反正还不是在作秀。"

****

![8-一厢情愿.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/动之以情/8-一厢情愿.png){ align=right width="20%" }
### 一厢情愿  
**认为一件事是真的或假的, 仅仅因为你情愿想当然。**  
"主席是不会犯错的。他是人民的领袖, 红旗的舵手。"

****

## 错误推论

![1-钦事证据.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/1-钦事证据.png){ align=right width="20%" }
### 钦事证据  
**对系统性研究下得出的证据视而不见, 反而集中在手头的个例上。**  
"我才不戒烟呢。我爷爷每天抽40根, 还活到了90岁！"

****

![2-合成谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/2-合成谬误.png){ align=right width="20%" }
### 合成谬误  
**推断一群人的特性或信条也代表了整个团体。**  
"最近的恐怖袭击是由激进的伊斯兰教徒组织的。因此所有的恐怖分子都是穆斯林。"

****

![3-分割谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/3-分割谬误.png){ align=right width="20%" }
### 分割谬误  
**将整个团体的特性或信条自动代入到每一名成员的头上。**  
"苹果的产品向来颠覆传统, 设计一流。下一款也一定如此。"

****

![4-设计谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/4-设计谬误.png){ align=right width="20%" }
### 设计谬误  
**因为某样东西设计精美, 视效上佳, 所以更加站得住脚。**  
"呃..."

****

![5-赌徒谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/5-赌徒谬误.png){ align=right width="20%" }
### 赌徒谬误  
**认为历史结果会影响未来结果。**  
"我已连续丢了10次硬币, 都是正面朝上。因此下一次更可能丢出反面来。"

****

![6-轻率概化.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/6-轻率概化.png){ align=right width="20%" }
### 轻率概化  
**从单一的样本得出概括性的结论。**  
"我被前面的女驾驶别了下。女人就是不能开车。"

****

![7-妄下定论.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/7-妄下定论.png){ align=right width="20%" }
### 妄下定论  
**没有公平考虑所有相关(且易举证的)事实, 就妄下结论。**  
"她想要医疗保险报销避孕药? 真是个婊子。"

****

![8-中间立场.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/8-中间立场.png){ align=right width="20%" }
### 中间立场  
**相冲突的两个观点似乎都有道理, 那么答案一定在两者的中间地带。**  
"我追尾了你的车, 但我不认为自己该出修理费。你认为我该出所有的修理费。合乎情理的方案就是平分费用。"

****

![9-完美主义谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/9-完美主义谬误.png){ align=right width="20%" }
### 完美主义谬误  
**认为只有完美的成功才是可行的选择, 从而反对任何低于预期的方案。**  
"这反酒驾的宣传究竟有什么用? 人们还是会醉酒驾车的。"

****

![10-相对论谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/10-相对论谬误.png){ align=right width="20%" }
### 相对论谬误  
**否定某样客观事实, 认为事实是相对一个或一群人而言的。**  
"那对你来说可能是对的。但对我来说不是。"

****

![11-以偏概全.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/11-以偏概全.png){ align=right width="20%" }
### 以偏概全  
**认为从小样本观察到的同样适用于整体。**  
"这家大型制鞋商在血汗工厂里雇用童工。可想而知所有制鞋公司都是邪恶的童工奴隶主!"

****

![12-一概而论.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/12-一概而论.png){ align=right width="20%" }
### 一概而论  
**宽泛地应用一般性原则。**  
"那些年轻人暴乱是因为他们缺失有道德观念的父亲。"

****

![13-中词不周延.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/错误推论/13-中词不周延.png){ align=right width="20%" }
### 中词不周延  
**因两件事有一个共通点, 那么他们就是同一回事。**  
"理论是尚未证实的观点。科学家用'进化论'这一词, 可见进化是未被证实的。"

****

## 操纵内容

![1-临阵救援.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/1-临阵救援.png){ align=right width="20%" }
### 临阵救援·  
**通过不断修改论据, 搪塞问题, 来保全自己的一贯主张。**  
"...但除了更好的卫生, 医药, 教育, 灌溉, 公共卫生, 道路, 净水系统和公共秩序...罗马人为我们做了什么？"

****

![2-一孔之见.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/2-一孔之见.png){ align=right width="20%" }
### 一孔之见  
**用不具代表性的样本所得出的结论, 来支持自己的论点。**  
"我们的网上调查表示, 90%的互联网用户反对网络隐私法。"

****

![3-确认偏误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/3-确认偏误.png){ align=right width="20%" }
### 确认偏误  
**挑拣对自己有利的证据, 而故意无视相冲突的。**  
"很明显911事件是美国政府为了合理化伊拉克与阿富汗战争而发动的阴谋。没有飞机撞上五角大楼。双子塔的倒塌是控制爆破...等等"

****

![4-伪二分法.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/4-伪二分法.png){ align=right width="20%" }
### 伪二分法  
**隐藏其它可能性, 将两个对立的观点看作仅有的选择。**  
"我们要么就得削减教育预算, 要么就得负更多的债。我们不能负更多的债了, 所以我们非降低教育预算不可。"

****

![5-谎言.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/5-谎言.png){ align=right width="20%" }
### 谎言  
**彻头彻尾的谎言, 被作为真相一再重复。**  
"我没有和那个女人发生性关系。"

****

![6-误导性解活个案.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/6-误导性解活个案.png){ align=right width="20%" }
### 误导性解活个案  
**用生动的细节来描述一次小概率事件, 以让别人相信这是一个问题。**  
"在法院判决同性婚姻合法化之后, 学校图书馆被要求存有同性文学作品;小学生会读到同性恋的童话故事, 甚至有明确支持同性恋的手册。"

****

![7-转移注意.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/7-转移注意.png){ align=right width="20%" }
### 转移注意  
**将毫不相关的话题引入辩论, 以干扰视线并导向不同的结论。**  
"参议员不需要为他开销的异常做出说明。毕竟, 有些参议员做的破事儿比这严重多了。"

****

![8-滑坡谬误.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/8-滑坡谬误.png){ align=right width="20%" }
### 滑坡谬误  
**认为开始的一小步会无可避免地引发一串相关(负面)的事件。**  
"如果我们将大麻合法化, 更多的人就会开始吸食毒品和海洛因。到时候我们就得也合法化那些。"

****

![9-隐瞒证据.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/9-隐瞒证据.png){ align=right width="20%" }
### 隐瞒证据  
**有意不用相关且重要的信息, 因为那对立于自己的结论。**  
"炒菜产生的油烟是PM2.5的重要来源, 所以要治理雾霾, 中国人就得少做菜。"

****

![10-无法证伪.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/操纵内容/10-无法证伪.png){ align=right width="20%" }
### 无法证伪  
**提出一个无法被证伪的观点, 因为无法加以验证。**  
"他撒谎是因为鬼上身了。"

****

## 混淆因果

![1-肯定后件.png](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/1-肯定后件.png){ align=right width="20%" }
### 肯定后件  
**认为对你所观察到的现象只有一种解释**  
"婚姻会带来孩子的降生。所以这就是其存在的理由。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/2-循环逻辑.png){ align=right width="20%" }
### 循环逻辑  
**论证的前提里已经蕴含结论**  
"《圣经》上说上帝存在。由于圣经是上帝的话, 圣经必然正确。所以上帝是存在的。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/3-相关即因果.png){ align=right width="20%" }
### 相关即因果  
**认为两个一起发生的事件一定有因果关系。(关联性=因果性)**  
"小混混们听主题暴力的饶舌音乐。所以饶舌音乐会造成青少年的暴力行为。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/4-否定前件.png){ align=right width="20%" }
### 否定前件  
**有这样的结果并非只有一个解释。因此, 在这样的情况下从结果反推原因是不准确的。**  
"如果你读了好学校, 你就会找到好工作。如果你没读好学校, 你就找不到好工作。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/5-忽视主因.png){ align=right width="20%" }
### 忽视主因  
**声称是某事件导致了后果, 而实际上另一件(意料之外)的事才是原因。**  
"60年代我们开始了性解放运动, 而现在人们正死于艾滋。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/6-前后即因果.png){ align=right width="20%" }
### 前后即因果  
**因为一件事是在另一件事之后发生的, 因此也是由那件事引起的。**  
"总统上台之后, 失业人数创了历史新高。所以总统阻碍了经济发展。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/混淆因果/7-积非成是.png){ align=right width="20%" }
### 积非成是  
**认为一桩错事能被另一桩错事所抵消**  
"不错——这监狱环境恶劣又没人性, 不过关的本来就是罪犯！"

## 发起进攻

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/1-人身攻击.png){ align=right width="20%" }
### 人身攻击  
**绕开论证, 针对辩论者本身发起不相干的攻击。**  
"你以为自己是生物学专家吗, 也好意思来教我们转基因食品的事？"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/2-举证责任.png){ align=right width="20%" }
### 举证责任  
**我不需要证明我说的正确——你必须证明它是错的。**  
"我坚持认为长期的太阳活动周期是全球变暖的原因。证明我错了啊。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/3-身份主观.png){ align=right width="20%" }
### 身份主观  
**认为一个论断不可信, 因为支持者与之有利益关系。**  
"研究手机对健康影响的这个调研有手机公司参与。所以, 研究结果不可信。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/4-基因谬误.png){ align=right width="20%" }
### 基因谬误  
**攻击一个论点的来源，而非它的内容。**  
"这本书是1967年出版的, 里面说的东西哪还有价值。"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/5-罪恶关联.png){ align=right width="20%" }
### 罪恶关联  
**通过将一个论点与形象不良的人或群体联系起来, 从而破坏其可信度。**  
"哦, 你想要放松反恐条例, 就像那帮恐怖分子想要的一样。所以你是支持恐怖主义的啰？"

****

![](https://cdn.jsdelivr.net/gh/ipaperclip-icu/static/image/诡辩术/发起进攻/6-稻草人谬误.png){ align=right width="20%" }
### 稻草人谬误  
**歪曲或简化你对手的论点, 以攻击之。**  
甲: "国家应该投入更多的预算来发展教育行业。"  
乙: "你这么不爱国, 居然想减少国防开支, 让外国列强有机可乘。"
